# （附加测试）环境可靠性测试（DDR DFS 2666/1333/333变频）
## 测试说明
系统需要使能DDR DFS功能，默认系统不使能该功能。
DDR DFS功能的开发需要针对开发板上的ddr训练多组ddr频点，不建议使能。
## 测试方法
1. 进入`11-ddr_dfs_test`目录。
2. 执行`sh ddr_test.sh &`将主动变频挂后台运行。
3. 脚本中包含`stressapptest`和`memtester`运行命令，可通过修改`-s`参数修改运行时间。
## 测试标准
1. 高温：45°、低温：-10°、常温下，程序正常执行，不会出现重启挂死的情况。
2. LOG中没有`fail`、`error`、`timeout`等异常打印。
3. 能稳定运行48小时。
4. 运行过程中需要关注CPU、DDR占比，CPU占比均接近90%，DDR带宽占比接近60%。
