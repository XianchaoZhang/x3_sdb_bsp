4.3.1. 概述

X3系统软件基于Linux操作系统，系核采用Linaro的Linux内核ARM分支，内核版本为：4.14。系统为应用运行的环境提供基础的C和C++库，硬件操作库，C++算法应用程序框架。

根据不同的产品形态，系统提供不同的软件模块，构成最终产品。
